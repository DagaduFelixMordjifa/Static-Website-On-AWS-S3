
# STEP 1: 
#### Log into your AWS Console
Navigate to S3 and click on create bucket to create a bucket with a unique name

# STEP 2:
#### Open your newly created bucket and upload your files. 
#### (visit: https:// free-css.com to download other static  website)

# STEP 3:
#### Go to the bucket Properties
#### Click on static website hosting and select Enable
#### Enter index document ( index.html)
#### Enter error document ( error.html)
#### Click Save

# STEP 4:
#### Setting Bucket Permissions
#### Go to Bucket Policy, Edit and Add Permission
#### Replace Bucket-name with your Bucket-name
#### Click Save
